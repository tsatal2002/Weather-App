My Weather App

This is a Weather App project for TECHPRO Academy, January 22 2026 18:00-21:00. You type the city name on the search bar that you want to know the weather of.

You run it this way

First Download the code:
   git clone <Your-Repo-Link>
   cd weather-app

Second Install the libraries:
   npm install

Third Start the app:
   npm run dev

How to setup it up the first time

You need an API key from OpenWeatherMap
Create a .env file and add this:

VITE_WEATHER_API_KEY=your_key_here

What features you will see?

Search: Find weather for any city.
Favorites: Save cities you check often.
Units: Switch between Celsius and Fahrenheit.
Dark Mode: It has a dark and light theme.
